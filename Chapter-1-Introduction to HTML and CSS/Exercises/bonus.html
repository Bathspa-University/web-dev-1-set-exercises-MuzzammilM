HTML & CSS Quick Reference Guide

HTML Tags

Here are some commonly used HTML tags:

  • <h1> to </h1> - Heading 1
  • <p> to </p> - Paragraph
  • <a> to </a> - Anchor (link)
  • <img> - Image
  • <ul> to </ul> - Unordered List
  • <ol> to </ol> - Ordered List
  • <li> to </li> - List Item
  • <div> to </div> - Division
  • <span> to </span> - Inline Span
  • <form> to </form> - Form
  • <input> - Input

CSS Properties

Here are some commonly used CSS properties:

  • color - Sets the color of text
  • background-color - Sets the background color of an element
  • font-size - Sets the size of the font
  • font-family - Sets the font of the text
  • margin - Sets the margin of an element
  • padding - Sets the padding of an element
  • border - Sets the border of an element
  • width - Sets the width of an element
  • height - Sets the height of an element

 <p>This is a paragraph.</p>

 <ul>

 </ul>
 <div>This is a division.</div>
